# Go Peerflix

A Golang port of [peerflix](https://github.com/mafintosh/peerflix).

## Installation
```sh
go get github.com/Sioro-Neoku/go-peerflix
```

## Usage
```sh
go-peerflix [magnet url]
```

To start playing in VLC (OsX only):
```sh
go-peerflix -vlc [magnet url]
```
